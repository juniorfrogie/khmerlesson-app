# Color Palette

## Primary

Primary dark blue:
- Main brand color
- Used for buttons, active states, headers

Suggested tones:
- #0F172A
- #1E3A8A

## Neutral Colors

Background:
- white
- light gray

Text:
- dark gray
- medium gray

Borders:
- soft gray

Status Colors:
- success
- warning
- error

Do not hardcode colors inside components.

All colors must come from theme tokens.
