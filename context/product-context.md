# Product Context

This app helps users learn Khmer vocabulary.

## Core User Flow

Open app
-> Browse courses
-> Open lessons
-> Study vocabulary flashcards
-> Continue progress

## Monetization

- Free courses available without login
- Premium courses require purchase
- Access valid for 1 year

## Learning Experience

- Vocabulary-focused
- Khmer -> English translation
- Pronunciation support
- Flashcard interaction

## Authentication

- Login (email, password), or GoogleAuth, or AppleAuth
- Register (email, password), or GoogleAuth, or AppleAuth
- Forgot password (email)
- Reset password (token, password)
- Verify apple id token

## Sync Strategy

- Sync service
- SQLite
- Zustand
